Block Elements:

Block Elements are the elements that take the entire width of the page and do not let any other elements come in their width. The Following are the commonly used Block Elements:

  1. <p> </p>
  2. <div> </div>

    3. And many more...

Examples:

HTML code

<p> This is a Block Element </p>

Css code

p{ color:black; background-color: cadetblue; }

This is a Block Element

From this you can see that the background color takes the entire width,so this shows a Block element.

Inline Elements:

Inline elments are the elements that only takes the space they need.The following are the commoly used Inline Elements:

  1. <a> </a>
  2. <span> </span>

    3. and many more...

Example:

HTML code

< a href="http//google.com" target="_blank" >Google< /a >

a{ color:blue; background-color: aqua; }
